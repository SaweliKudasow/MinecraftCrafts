# Minecraft Crafts

Веб-приложение для просмотра рецептов крафта на верстаке 3×3 в **Minecraft Java Edition**. Поиск по предметам, наглядная сетка верстака и иконки из официальных текстур.

Данные предметов и рецептов загружаются из [minecraft-data](https://github.com/PrismarineJS/minecraft-data) (версия **1.20.5**). В списке отображаются только предметы и блоки, для которых есть хотя бы один рецепт обычного верстака.

## Возможности

- Поиск по английскому имени или отображаемому названию (`diamond`, `oak_planks` и т.д.)
- Отображение рецепта в сетке 3×3 с подписью «бесформенный крафт», если применимо
- Несколько вариантов рецепта для одного предмета (переключение кнопками ◀ / ▶)
- Иконки предметов из [Minecraft Item Gallery](https://minecraftallimages.jemsire.com) с запасными источниками текстур
- Интерфейс в стиле Minecraft (шрифт, тёмная тема, подсказки при наведении)

## Требования

- [Node.js](https://nodejs.org/) 18 или новее
- npm (идёт вместе с Node.js)
- Подключение к интернету — данные и текстуры подгружаются с GitHub и CDN при работе приложения

## Установка и запуск

Клонируйте репозиторий и перейдите в папку проекта:

```bash
git clone <url-репозитория>
cd MinecraftCrafts
```

Установите зависимости:

```bash
npm install
```

Запустите режим разработки (по умолчанию [http://localhost:5173](http://localhost:5173)):

```bash
npm run dev
```

## Сборка для продакшена

Собрать статические файлы в папку `dist/`:

```bash
npm run build
```

Локально посмотреть production-сборку:

```bash
npm run preview
```

## Стек

- [Vite](https://vitejs.dev/) — сборка и dev-сервер
- Vanilla JavaScript и CSS, без фреймворков
- Внешние данные: PrismarineJS/minecraft-data, MinecraftAllImages, minecraft-assets
